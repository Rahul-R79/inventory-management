# Quick Start Guide

## To use the Interactive CLI:

```bash
npm run dev
```

Then follow the on-screen menu to:
- Add items
- View all items
- Search by ID
- Update item details
- Update stock
- Delete items

## To run the demo:

```bash
npm start
```

This will automatically demonstrate all CRUD operations.
